# CampusMate User Manual

CampusMate helps students manage assignments, deadlines, study sessions and campus reminders.

## Getting started

1. Open the app.
2. Continue as guest or sign in with an email account.
3. Use the bottom navigation to move between Home, Tasks, Study, Analytics and Settings.

## Home

The Home screen shows a quick overview of upcoming work, urgent deadlines and device context. Use this screen to decide what to work on next.

## Tasks

The Tasks screen lets you search, filter and complete study tasks.

To create a task:

1. Open Tasks.
2. Tap Add Task.
3. Enter a title, subject, deadline and priority.
4. Save the task.

To edit a task, open its detail page and choose Edit.

## Location reminders

When creating or editing a task, open the location reminder option. Select a campus place, confirm the map location and choose a reminder radius.

## Study timer

Open Study, choose a task or subject, then start the timer. Pause when needed and save the session when finished. Saved sessions appear in Analytics.

## Analytics

Analytics shows weekly study time, completed tasks and subject balance. Use the audio summary button to hear a short spoken progress update.

## Settings

Settings includes:

- dark mode
- text size
- accessibility mode
- notification controls
- background reminder controls

## Troubleshooting

| Problem | Suggested action |
| --- | --- |
| Notifications do not appear | Check notification permission and use a development build for full testing |
| Location does not update | Check location permission and device GPS settings |
| Text feels too small | Increase text size in Settings |
| Data looks reset | Check whether guest mode/local storage was cleared |
